# **App Name**: ProjectWise

## Core Features:

- Document Management: Centralized repository for project documents with controlled access, versioning, and metadata. Support drawings, models and reports.
- Project Communications: Structured requests for information (RFIs), transmittals, and site instructions, linked documents and responses, and conversation history retention.
- Workflow Management: Configurable review and approval workflows with deadline tracking and action ownership.
- Audit Trail: Time-stamped records, user and company attribution, and response tracking.
- Role-Based Content Summaries: AI tool which prepares a summary document which pulls only the most pertinent content based on a pre-defined understanding of what a specific user persona finds most valuable. Each persona should be selectable from a UI.

## Style Guidelines:

- Primary color: Dark teal (#008080) to convey professionalism and trust.
- Background color: Light gray (#F0F0F0) to ensure readability and a clean interface.
- Accent color: Gold (#FFD700) for highlighting key actions and elements.
- Headline font: 'Poppins' sans-serif for a modern, geometric look.
- Body font: 'PT Sans' sans-serif for a readable and clean design.
- Use clear and professional icons to represent document types, actions, and roles.
- Modular, scannable sections with clear CTAs, inspired by monday.com.
